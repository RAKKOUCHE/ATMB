Detailed Description

Exceptions and error code raised by the API.

Access to any method or property can result in an exception. There is 3 groups of exceptions:

  1. API error - This is errors related to the API itself, like missing dll, USB driver, API not openned before to use its methods, etc... This category of errors is defined in Mei::Bnr::BnrApiErrorCode.
  2. USB error - This is all errors related to the USB communication. This category of errors is defined in Mei::Bnr::BnrUsbErrorCode.
  3. XFS error - Errors related to the XFS protocol used by the BNR. This category of errors is defined in Mei::Bnr::BnrXfsErrorCode.
