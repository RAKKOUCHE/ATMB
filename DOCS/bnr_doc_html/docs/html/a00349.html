void Mei::Bnr::Module::Module::RevertToBoot ( )

Erases the main application of the Module.

In case of the MainModule, this command erases also the Billset and the CashUnit::LogicalCashUnit.

Exception:
  • Illegal - The BNR is in a state where it cannot execute the operation.
  • CashInActive - A transaction started with Bnr::Bnr::CashInStart()
  • NotSupported - The property is not suppoted by the current firmware.
  • TooManyBills - No transaction is active, but the BNR has bills in escrow, or in its transport system.
  • Failure - The command un-succeeded due to an internal failure.
See also
BnrXfsErrorCode, BnrApiErrorCode, BnrUsbErrorCode for more exception details.
