Detailed Description

Bnr class is the main class to access and drive BNR system.

It provides all methods and propoerties to do bill transactions. Bnr configuration is achieved by using the Bnr class, Module class and its sub-classes.

void Mei::Bnr::Bnr::UpdateCashUnit ( CashUnit::CashUnit^  cashUnit)

Updates the BNR's cash unit.

This function is used to change counts and thresholds of the BNR cashUnits.

Those settings are persistent over power cycles.

Parameters
[in,out]cashUnitNew cash unit. On return, the cashUnit instance contains only the cashUnits that were updated.
