BNR API for .NET Documentation
Introduction
The present document is intended for Integration Software Engineers of OEM's (Original Equipment Manufacturer) integrating a MEI "Banknote Recycler" ("BNR") into an automated payment machine.
This document contains:
  • a description of the implementation of the "Banknote Recycler"
  • the principles for the use of the .Net "Banknote Recycler" API
  • the necessary information for the management
  • reference data of the .Net "Banknote Recycler" API
Supported platforms
Windows 32-bit and 64-bit platforms are supported. Frameworks .NET 2.0 (BnrCtlDotNet.dll) and .NET 4.0 (BnrCtlDotNet40.dll) are supported.
Getting Started
To begin reading the API documentation, start with the BNR Concepts which is a good introduction to the "Banknote Recycler" API. Then, Modules page will give you an overview of the API interface. Finally, the Namespaces List will be very usefull to find more specific details.
