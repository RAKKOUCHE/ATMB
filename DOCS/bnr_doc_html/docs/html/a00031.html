int Mei::Bnr::CashUnit::LogicalCashUnit::Number
getset

The unique number of cash unit.

This number is assigned (or reassigned) on Bnr.Reset() and identifies the unit along the time; therefore, it can be used to track unit changes, or uniquely reference units in method calls (DenominationItem.unit property is an example).
