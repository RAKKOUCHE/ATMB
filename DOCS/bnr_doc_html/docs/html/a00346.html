Enumeration Type Documentation

enum Mei::Bnr::HistoryReportReason
strong

Reason for creation of a HistoryReport.

Firmware Compatibility
Introduced in BNR Firmware v 1.7.0.
See also
Report::HistoryReport::Reason.
Enumerator
NrOfCyclesReached 

The system has done 10'000 cycles since the last HistoryReport.

ModuleChange 

A Spine or Recycler module has been replaced.

FirmwareChange 

A new application firmware has been downloaded (in Main Module or other modules).

BillsetChange 

Change or update in the downloaded denominations has been detected.

Manual 

A command Bnr::Bnr::SaveHistoryReport() was sent to the BNR.

MaintenanceDone 

A command Module::MaintenanceDone() was sent to the BNR.
