void Mei::Bnr::Module::Module::MaintenanceDone ( )

Indicates to the module that the maintenance operation was done (reset of MaintenanceInfo::CycleSinceLastMaintenance counter and set MaintenanceInfo::MaintenanceStatus to Ok)

Exception:
  • ParameterInvalid - The module is not valid.
  • Illegal- The state of the BNR does not allow this command (e.g. asynchronous command running, cash module lock open, etc...).
  • Busy - A transaction is in progress.
  • NoHardware - The module is missing.
  • Failure - The command un-succeeded due to an internal failure.
  • NotSupported - The command is not suppoted by the current firmware.
StatusEvent event:
The field status can take the following values :
  • MaintenanceStatusChanged - if the command call modifies the MaintenanceInfo::MaintenanceStatus.
Firmware Compatibility
This function requires a BNR FW v1.7.0 or newer. With older FW versions, a NotSupported exception will be raised.
