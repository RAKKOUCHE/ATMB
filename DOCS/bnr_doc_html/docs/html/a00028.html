int Mei::Bnr::CashUnit::DepositCounters::emptiedCount

Number of bills in this cash unit that were in the transport system while a BNR door was opened and then were stored in the cashbox on the subsequent Bnr.Reset().

Note
Due to the fact that those bills were accessible to an operator, this count cannot be guaranteed to be accurate.
